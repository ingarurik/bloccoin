🎮 Rejoindre

Prêt à bâtir votre légende ?

Lancez Minecraft Java Edition

Assurez-vous d'avoir Minecraft Java Edition installé et mis à jour. Bloc-Coin est compatible avec les versions récentes du jeu.

Accédez à "Multijoueur"

Depuis le menu principal, cliquez sur Multijoueur puis sur Ajouter un serveur.

Entrez l'adresse du serveur

Dans le champ "Adresse du serveur", saisissez :

play.bloccoin.fr
Rejoignez et explorez !

Connectez-vous et commencez votre aventure. Un guide d'accueil vous attendra sur l'île de départ pour vous orienter dans ce monde suspendu.
